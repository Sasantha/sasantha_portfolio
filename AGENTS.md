# Project: Sasantha Portfolio (Next.js)

## Goal
Build a clean, fast portfolio website for a full-stack developer.
Must be deployable to Vercel.

## Tech
- Next.js App Router
- TypeScript
- Tailwind CSS

## Pages
- / (Home)
- /projects (all projects)
- /projects/[slug] (project detail)
- /contact (simple contact + links)

## Content source (for now)
Use local data in: /content/projects.ts and /content/profile.ts
Later we will replace with an admin panel.

## UI requirements
- Modern, minimal, mobile-first
- Sticky navbar, section anchors, CTA buttons
- Project cards with: title, stack, summary, links (live/repo), tags
- SEO: metadata, OpenGraph, accessible headings
